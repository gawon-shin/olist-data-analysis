# Olist E-commerce 데이터 분석

## 1. 프로젝트 정보

### 프로젝트 목적
이커머스 데이터를 기반으로 매출, 배송, 재구매 현황을 분석

### 데이터셋
- Kaggle Olist 브라질 이커머스 데이터
- 2016~2018 / 주문 112,650건 / 고객 99,441명

### 사용 기술
| 기술 | 용도 |
|------|------|
| SQL | 데이터 추출 및 집계 |
| Python (Pandas) | 데이터 가공 및 구간화 |
| Power BI | 시각화 |

<br>

## 2. 분석 내용

### Q1. 월별 주문 수와 매출은 어떻게 변했을까?

**분석 의도** : 매출 흐름과 성장 추이를 파악하기 위해

**분석 방법** : SQL로 월별 주문 수, 매출, 평균 객단가(AOV)를 집계

**결과**
- 2017년 하반기부터 주문 수, 매출 급성장
- AOV는 130~140 BRL 구간에서 안정적 유지
- 2018년 이후 매출·주문 수 모두 소폭 감소

**해석**
- 매출 성장은 가격 상승이 아닌 주문 수 증가에 의한 것
- 특정 시점 급증 → 이벤트/프로모션 영향 가능성
- 2018년 이후 성장 둔화

<br>

### Q2. 매출은 특정 상품/카테고리에 집중되어 있을까?

**분석 의도** : 매출 구조가 특정 상품·카테고리에 집중 혹은 분산되어 있는지 파악하기 위해

**분석 방법** : SQL로 상품·카테고리별 매출 및 비중 산출

**결과**
- 상위 10개 상품 매출 비중 3.35%
- 상위 5개 카테고리 매출 비중 39.8%

**해석**
- 상품 단위에서는 매출이 분산된 구조 (히트상품 없음)
- 카테고리 단위에서는 일부 집중되지만 특정 카테고리 독점 구조는 아님

<br>

### Q3. 배송 소요 시간은 어느 정도 수준일까?

**분석 의도** : 배송 기간과 지연 수준을 파악하기 위해

**분석 방법** : SQL로 배송 소요일 및 지연 여부 산출, Pandas로 배송 구간화

**결과**
- 평균 배송 소요일 12.5일
- 6-10일(34.4%), 11-20일(34.7%) 구간이 대부분
- 배송 지연 비율 8.1%

**해석**
- 표준 배송 중심 구조 (초고속·초지연 비중 낮음)
- 20일 초과 배송(13.6%)이 존재하나, 예정일 기준 지연 비율은 8.1%로 낮은 수준

<br>

### Q4. 고객의 재구매는 어느 정도 수준일까?

**분석 의도** : 재구매 고객 비율과 매출 기여도를 파악하기 위해

**분석 방법** : SQL로 재구매 고객 및 매출 집계, Pandas로 재구매 기간 구간화 (당일 재구매 제외)

**결과**
- 재구매 고객 비율 2.2%, 매출 기여 4.1%
- 전체 고객 98%가 1회 구매에 그침
- 평균 재구매 기간 115.3일
- 91일 이상 구간 45.3%로 가장 높음

**해석**
- 재구매율과 매출 모두 낮아 고객 유지 구조 약함
- 재구매해도 장기 이탈 후 복귀 패턴 중심

<br>

## 3. 결과 정리

| 항목 | 요약 |
|------|------|
| Q1 매출 추이 | 주문 수 기반 성장 구조, 2018년 이후 성장 둔화 |
| Q2 매출 구조 | 상품 단위 분산 + 카테고리 단위 일부 집중 |
| Q3 배송 현황 | 예정일 내 배송이 대부분 → 배송 안정성 확보 |
| Q4 재구매 현황 | 재구매율 낮고, 재구매까지 평균 4개월 소요 → 신규 유입 의존 구조 |
